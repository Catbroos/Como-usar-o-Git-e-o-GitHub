GitHub Pages

O GitHub Pages é uma funcionalidade do GitHub onde podemos hospedar sites e criar links de forma gratis no GitHub.

Para isso primeiro abra o repositório do projeto.Clique em Settings.

Na barra lateral na parte Code and automation clique em pages.

Lá na parte de Branch colocamos main, root e depois save.Assim é só esperar e estará com seu link.Você também pode personalizar o link mas não vou entrar em detalhes pois isso é pago.

Para ter o link vá na raiz do projeto e clique em Deployments.
