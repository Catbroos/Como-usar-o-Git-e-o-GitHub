Configurando o Git

Você lembra do Git Bash lá do primeiro módulo?Bom agorá vou passar alguns comanos para configuralo.

  • "git config --global user.name "(seu nome)"":Esse comando seta nas configurações que o nome do usuário é tal;
  • "git config --global user.name "(seu email)"":Esse comando é a mesma coisa que o primeiro só que ele define o email e não o nome do usuário;
  • "git config --list": Esse lista todas as suas configuracões do Git;
  • "git config user.name":Esse mostra especificamente qual seu nome registrado;
  • "git config user.email":Esse mostra especificamente qual seu email registrado;

Agorá você precisa definir no Git Bash seu nome e email (Dica: se você já tem uma conta no GitHub coloque aqui o mesmo email que você definiu lá) usando os dois primeiros comandos.Depois verifique se o nome e o email estão corretos.
