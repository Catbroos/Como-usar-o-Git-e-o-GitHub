Git clone e git pull

Git clone

O git clone é um comando que pega um repositório no GitHub e faz uma cópia dele na pasta em que você está trabalhando.

A estrutura dele é assim: "git clone (link do repositório no GitHub).git".

Dica: Fazendo isso não precisamos dar os comandos iniciais de git init, git remote, e tal.Então podemos primeiro criar um repoisitório no GitHub e depois damos o git clone.Pois assim não precisamos dar os comandos iniciais.

Git pull

O git pull é um comando que copia todas as alterações que estão no repositório do GitHub e cola no seu repositório local.

A estrutura do comando é assim: "git pull"(é só isso pois o repositório local já deve estar linkado com o on-line.)
