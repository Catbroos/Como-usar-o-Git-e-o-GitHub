Tokens pessoais do GitHub

Agorá nessa tela:

Temos que infor para o GitHub que somos nós mesmos.Para isso temos alguns jeitos.A qui vou mostrar o modo mais fácil.

Tokens pessoais são um jeito de que o GitHub saiba que você é você.Para fazelos:

1:Abra sua conta do GitHub, e clique em sua foto de perfil.E depois settings.

Na barra lateral clique na última opção Developer settings.

Clique em Personal access tokens e depois Tonkens(classic).

Clique em generation new token e nas duas opções escolha generation new token(classic).

Agora é só colocar as informações solicitadas que são o "nome" do token, sua data de expiração, e os escopos que nesse caoso você pode asinalar o primeiro.

Clique em Create new token e pronto agorá todo projeto que você fizer na primeira vez que for dar o "git push -u origin main" aparecerá aquela tela e é só colocar para ir com o token colocalo lá e pronto.

Outra coisa.Apartir da primeira vez que der o "git push -u origin main" e prencher tudo.Das proximas vezes na quele projeto só terá que ser dado o comando "git push".
