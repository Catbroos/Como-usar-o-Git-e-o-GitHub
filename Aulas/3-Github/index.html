GitHub

GitHub não é a mesma coisa que o Git.O GitHub é uma rede social de programadores que você pode ver códigos de outras pessoas e melhorlo.Lá em um lugar que os recrutadores olham quando vão contratar alguém, eles olham várias coisas como: se você é ativo lá, tem vários projétos, contribui no código dos outros e entre outras coisas.

Alem disso é posivel subir projetos simples para web e criar um link para outros acessárem.Normalmente se você passar para a próxima fase da recruta os recrutadores te pedem para fazer um projeto tal e subir no seu GitHub.

Para acessar o GitHub clique neste link https://github.com/.
