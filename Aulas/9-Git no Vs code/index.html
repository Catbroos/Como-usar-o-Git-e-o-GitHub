Atalho do Git no Vs code

O Vs code tem uma funcionalidade muito interesante e legal.

Na barra lateral tem uma parte chamada controle de versão, que é como dar git add, git commit, git push e tal.

Ele separá os arquivos Untracked e Modified em "Changes" dos arquivo Staged em "Staged Changes".Só que ele não mostra os arquivos Unmodified.

Em "Changes" aparece os arquivos que estão Untracked com umU verde e os Modified com um M roxo.

Os arquivos Staged ficam com um A verde saturado.Se você modificar um arquivo e depois dar o git add, ele fira com um M em amarelo, que também podem ser commitados.

Com o muse em cima do arquivo aparece um + que é o mesmo que usar o git add.Clicando no arquivo ele mostra as alterações que foram feitas nele.Tem uma caixa de texto onde seria a mensagem do commit preenchendo-a e clicando no botão logo abaixo escrito Commit damos o commit.Para dar o push podemos clicar nos três pontos em Souce Control e push.

Fizemos tudo da aula anterior só que com a interface grafica do Vs code.Só temos que fazer a parte inicial com os comandos mesmo.

Dica: Se você não decorou os comandos eu indico que faça isso pelos comandos, podendo consultar anotações e tal.Se você decorou e já sabe pode usar a interface grafica!
